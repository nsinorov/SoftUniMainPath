Write a function that receives a grade between 2.00 and 6.00 and prints a formatted line with grade and description.

     < 3.00 - "Fail"
     >= 3.00 and < 3.50 - "Poor"
     >= 3.50 and < 4.50 - "Good"
     >= 4.50 and < 5.50 - "Very good"
     >= 5.50 - "Excellent"
