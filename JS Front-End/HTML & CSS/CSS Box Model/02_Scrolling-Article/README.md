### Requirements:

    HTML with <body> that contains: <section> with <article> which has <h2>, <h3> and <p>.

CSS:

    • <article>:
        o Width: 500px
        o Height: 200px
        o Background color: lightgray
        o Paddings: 10px
        o Auto overflow
        
    • <h2>:
        o All characters to be UPPERCASE
        o Align the text to the center
        
    • <h3>:
        o Align the text to the center
        
    • <p>:
        o Indent text in the first line 30px
